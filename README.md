# Jonathando.com

The personal site for Jonathando — builder, investor, creator.

## Stack

Plain HTML, CSS, and JavaScript. No build step. Open `index.html` in a browser or serve the directory with any static host (GitHub Pages, Netlify, Cloudflare Pages, Vercel, S3, etc.).

## Local preview

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## Structure

- `index.html` — markup and content
- `styles.css` — design system, layout, responsive rules
- `script.js` — scroll reveal + small interactions

## Deploying

Point your domain at any static host that serves the repo root. For GitHub Pages, enable Pages on the branch and Jonathando.com is live.
